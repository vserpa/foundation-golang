# foundation-golang
Golang foundation exercises

# Utils

## Converter
```shell
# go run utils/converter.go <values> <unity: kilometers|celsius>
go run ./utils converter 80 100 120 kilometers
```
## Infinite Loop
```shell
# go run utils/converter.go <values> <unity: kilometers|celsius>
go run ./utils infinite-loop
```
## Named Loop
```shell
# go run utils/converter.go <values> <unity: kilometers|celsius>
go run ./utils named-loop
```

# DSA

## Quicksort
```shell
# go run ./dsa quicksort <values>
go run ./dsa quicksort 3 2 9 0 8 4 7
```

## Maps
```shell
# go run ./dsa maps <words>
go run ./dsa maps gallery circle ground around Grammar
```
## Stack
```shell
# go run ./dsa stack
go run ./dsa stack
```
